<h1>todo App</h1>
<hr><p>it's simple full stack todo app , which is developed in react native in frontend and node js in backend and mongo db for the database</p><h2>General Information</h2>
<hr><ul>
<li>In this project we can categorize our todo list and also check the listing according to categories.</li>
</ul><ul>
<li>The goal of this app is to help us become more aware of how we spend time in the process of doing those tasks and how productive that time is. It can help set some constraints on social media to reduce distraction and track the time we spend working on the todo items.</li>
</ul><ul>
<li>Increases productivity, Repeating tasks ,Improves memory ,Prioritization , Provides motivation ,Sync ,Task management , Reorder tasks , Time tracking</li>
